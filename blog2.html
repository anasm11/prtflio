Callbacks, promises and async/await in javascript

Callback functions in javascript

In javascript, it is possible to pass a function as an argument to another function. The function which is passed as an argument is called callback function. The callback function can be called at anytime inside the function to which it is passed. This gives us the control to call it as per the need when required.

Example syntax of callback:
function func2(){
console.log('this is called back.')
}

function func1(callback){
console.log('this is main function.')
callback();
}

func1(func2);

In the above example, func2 is called back from func1. This is particularly helpful in a scenario when we have to operate on data which is not available currently and has to be fetched from somewhere else. A good example would be in the case of an api call, where the data has to be fetched from another server and it may take some loading time. Also, in the case of async finctions where one function has to wait for another function, callbacks are useful. In promises, callback functions can be useful when called in .then block at the fulfilment/rejection of a promise.


Promises

A promise in javascript is like a promise in real life - an assurance to get something done. It works as a way to handle something whose value is not known currently. A promise can have 3 states: pending, fulfilled or rejected. A pending promise eventually can be fulfilled or rejected. When it is fulfilled , the .then handlers associated with it are called. A handler with .catch is added at the end, which gets called when the promise is rejected.

Example syntax for promise:
const promise1= new Promise(function(resolve,reject){
setTimeout(function(){
resolve('resolved');
},1000);
})

promise1.then(handler1)
.then(handler2)
.catch(errorHandler);

In the above example, promise1 is a promise created using new keyword. The .then blocks have handlers which are chained with an ending .catch block.



Async/await in javascript

Async/await are an easier way for writing asynchronous code. Adding async in front of a function makes it asynchronous and returns a promise object. await keyword is generally used while calling asynchronous functions returning promise(which may take time and whose value isnt known currently). It pauses the execution till the result of promise is evaluated.

Example syntax for async/await keywords:
async function fun(){
try{
await promise1;
await handler1();
handler2();
}
catch(){
errorHandler();
}
}
fun();
